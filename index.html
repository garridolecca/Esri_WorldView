ESRI WORLDVIEW
SATELLITE INTELLIGENCE SIMULATOR
// OVERVIEW

Real-time global intelligence visualization combining satellite tracking, aircraft monitoring, seismic activity, natural disasters, submarine cable infrastructure, and military installations on a 3D globe.

// DATA FEEDS
  • Satellites — CelesTrak TLE / SGP4
  • Aircraft — OpenSky Network ADS-B
  • Earthquakes — USGS Real-Time
  • Natural Events — NASA EONET
  • Submarine Cables — ArcGIS
  • Military Bases — US DoD / ArcGIS
  • CCTV Cameras — OpenStreetMap / Overpass (city view)
// CONTROLS
  • Left-click + drag — Rotate globe
  • Right-click + drag — Pan
  • Scroll wheel — Zoom in/out
  • Click feature — View details
  • Render modes — NVG / Thermal / CRT
  • Layer toggles — Show/hide datasets
  • Search bar — Fly to any city in 3D
// RENDER MODES
STD Standard satellite imagery
NVG Night vision green phosphor
THM FLIR thermal imaging
CRT Retro CRT with scanlines
Built with ArcGIS Maps SDK for JavaScript 5.0 • satellite.js • Open Data APIs
